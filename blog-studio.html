Facebook & Instagram Auto-Posting
Get this from developers.facebook.com → your app → Graph API Explorer → generate a Page access token for your KAYAL Facebook Page.
Found on your Facebook Page → About → Page ID. Or go to your page URL and the number in the URL.
How to get your Page Access Token:
1. Go to developers.facebook.com
2. Click My Apps → Create App → Business type
3. Add "Facebook Login" and "Instagram Graph API" products
4. Go to Graph API Explorer → select your app
5. Select your Page from the dropdown
6. Generate token with permissions: pages_manage_posts, instagram_basic, instagram_content_publish
7. Copy the token and paste above
Facebook & Instagram Auto-Post

One-time setup required: You need a Meta access token to post automatically.
1. Go to developers.facebook.com/tools/explorer
2. Select your app → Generate Access Token
3. Add permissions: pages_manage_posts, instagram_basic, instagram_content_publish
4. Click Generate and paste the token below
